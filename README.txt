INSTALLAZIONE :

1) MySQL
2) MongoDB
3) XAMPP (o separatamente PHP ed APACHE WEB SERVER)
4) NodeJS

5)Inserire la cartella WalkToWork nella cartella  XAMPP/htdocs


CARICAMENTO AGENZIE 

1) Avvia server Node : caricamentoAgenzie.js (da terminale raggiungere la cartella in cui lo script è salvato e chiamare il comando da shell : node caricamentoAgenzie.js )
2) Avvia Script : localhost/Aquire/leggiJSON.php


CARICAMENTO BANDI 

1) Avvia lo script localhost/Acquire/ScrapeDBGazzetta.php (modificando in file_get_contents l'URL della pagina della gazzetta da caricare)


//////////////////////////////////////////////////////////////////////////////////////////////////////////////////////////

START :

1) Avvia server Node 
2) Avvia sendAgenzie.js 
3) Avvia Apache Web Server 
4) Dal browser digita localhost/WalkToWork/index3.html




Sezione Annunci : visualizzaAnnunci.php
Sezione Bandi : bandi.php
Sezione Agenzie : agenzie.php
Sezione Recensioni : homeRecensioni.php
